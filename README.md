# My-P-XuLie2XML2MID
一种自定义格式P序列与XML和MID互转程序
◆依赖：Python 3.14.5 win64版本
    官网：www.python.org
    安装后，cmd命令行运行：py
    输出：
    ◇Python 3.14.5
    为安装成功

◇依赖库：如果报错，检查输出信息，缺少库则安装对应的库，如果其它错误请检查文件格式释放是程序需要的格式
    可能需要：启用MAX_PATH，“启用Win32长路径”策略‌（组策略或注册表设 LongPathsEnabled=1）

◇几乎所有代码由文心AI给出，在基础上进行修正，如果报错，请检查对应库是否安装

◆”P序列“只是代称，只是一种自定义存储方法，没有名字

    ◇这里是任意格式曲谱转”P序列“的一种方案

◆“P序列”编码方式：由浮点数存储，解\编码时转为整数进行，低24bit存储了7bit音符编号+4bit曲调+13bit毫秒时间值
                 编号范围：1~47
                 音域：对应G2#~F6#，midi音符值44~90，相对偏移43
                 注意事项：32bit浮点数精度有限，只能保证这3字节是准确的

◇XML是XPosition格式，XPosition现已经无法使用

◆“P序列”在游戏中，由于目前原理限制只能用单声部音乐，所以需要打谱软件进行编辑曲谱文件

    ◇一些比较常用的：overture，musescore，everyone piano的简谱大师等
    ◇这里给出一些mid文件的来源方式
        可以自行在各大音乐网站获取：虫虫钢琴、everyone piano、弹琴吧、midishow、musescore等
        或wav转mid工具

◇也可在钢琴、键盘钢琴中弹奏录制mid文件，后进行mid转为P序列文件，导入游戏
    ◇键盘钢琴：everyone piano、freepiano

◆这里用mid转为XML
    ◇预处理mid文件：可以用相关mid编辑软件
    ◇当然也可以用这里的mid转XML，用文本编辑器手动修正
    ◇(拖放文件名最好不要有空格、英文引号)

◆实验
    ◇滤杂音、和弦、单声化：XML=>P序列=>mid=>mid转xml筛选=>P序列
        配合mid编辑工具，多次进行mid=>mid转xml筛选=>P序列=>mid过滤处理
        直至达到目标效果
        (按需修改MIDtoXML2筛选音符.py中的筛选：时长<N ms 且 力度n~N→ 跳过注释下的代码)
